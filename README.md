# Task 1: Web Scraping Using Octoparse

## Introduction

The objective of this task was to collect data from a public website using web scraping techniques and create a structured dataset for analysis.

## Tool Used

* Octoparse

## Data Source

* Books to Scrape (Public Practice Website)

## Methodology

1. Created a custom task in Octoparse.
2. Entered the website URL.
3. Configured pagination to navigate through multiple pages.
4. Applied loop item extraction.
5. Extracted relevant fields such as book title, price, stock status, product URL, and image URL.
6. Executed the scraping task and exported the results to Excel format.

## Extracted Data Fields

* Book Title
* Price
* Stock Status
* Product URL
* Image URL

## Results

* Total Records Extracted: 1000
* Output Format: Excel (.xlsx)

## Conclusion

The web scraping task was successfully completed using Octoparse. A structured dataset containing 1000 book records was created and exported for further analysis. The task demonstrated the use of web navigation, pagination handling, and data extraction techniques.
